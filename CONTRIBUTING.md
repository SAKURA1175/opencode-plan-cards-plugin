# Contributing

Thanks for contributing to `opencode-plan-cards-plugin`.

## Development setup

```powershell
git clone https://github.com/SAKURA1175/opencode-plan-cards-plugin.git
cd opencode-plan-cards-plugin
npm install
```

## Local checks

```powershell
npm run check
npm run build
```

## Pull request checklist

- Keep changes focused and minimal.
- Update docs when behavior changes.
- Add or update `CHANGELOG.md` entries under `Unreleased`.
- Ensure `npm run check` and `npm run build` pass.

## Commit style

Recommended format:

- `feat: ...`
- `fix: ...`
- `docs: ...`
- `chore: ...`

## Reporting issues

- Use GitHub Issues for bug reports and feature requests.
- Include environment details: OpenCode version, plugin version, OS, and reproduction steps.

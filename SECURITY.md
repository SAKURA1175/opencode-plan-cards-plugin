# Security Policy

## Supported Versions

Security fixes are provided for the latest published minor/patch release on npm.

| Version | Supported |
| --- | --- |
| 0.1.x | ✅ |
| < 0.1.0 | ❌ |

## Reporting a Vulnerability

Please do not disclose security issues publicly before a fix is available.

- Preferred: GitHub private vulnerability reporting (Security tab).
- Fallback: open a GitHub Issue and mark it clearly as a security concern without sensitive details.

Include:

- plugin version
- OpenCode version
- OS
- reproduction steps
- impact assessment

You can expect an initial response within 72 hours.
